Lets use python to scrape a simple webpage and read data.
We will be using a module/lib called 'scrapy' for the same.
Scrapy is an opensource web scraping framework that can be used 
to scrape data from websites.

We will scrape data from the following website:
http://quotes.toscrape.com/

Lets get started !!

1. Install the scrapy module: 'pip install scrapy'
2. Open the terminal or commapnd prompt at your project 
directory location
3. Create a new scrapy project using the following command:
'scrapy startproject MyScrapper'
4. Next go to the spider sub-directory and create a new spider
using the following command:
'scrapy crawl myspd'
